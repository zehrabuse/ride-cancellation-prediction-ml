## 1. Working System
Our system predicts ride outcomes using a logistic regression model.

Users can input ride details such as distance, vehicle type.
The system scales inputs and provides instant predictions with probability scores.
The main interface is a simple web form with submit and clear buttons.
Example feature: real-time prediction for entered ride details.
